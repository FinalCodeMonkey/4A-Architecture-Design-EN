# 4A Architecture Design Skill

A layered 4A enterprise architecture design SKILL for end-to-end blueprint design and alignment across **BA (Business Architecture) / DA (Data Architecture) / AA (Application Architecture)**.

## Repository Purpose

- Provides 4A architecture layering methodology and naming standards
- Supports N:1:1 derivation from value chains to Business Objects (BOs) and services
- Applicable to scenarios such as MTL (Market to Lead), LTC (Lead to Cash), and CSM (Customer Success Management)

## Keywords

- 4A, EA, Enterprise Architecture, Business Architecture, Data Architecture, Application Architecture
- BA, DA, AA, Capability Map, Value Chain, Domain Modeling, BO, Service Design, API Design
- Layered architecture, three-layer alignment, N:1:1 derivation
